Asignamos a la variable URL de antes, de donde pretendemos sacar los datos, para que Python pueda reconocerla

url = "https://api.covid19api.com/countries"

Comparacion

Para comparar los datos de los tres países decidimos hacerlo en un solo gráfico

Primero creamos una variable distinta para cada uno de los dataframe de los países que hemos escogido: España, Italia y Alemania

casos_es = df_es.set_index("Date")["Cases"]
casos_it = df_it.set_index("Date")["Cases"]
casos_ger = df_ger.set_index("Date")["Cases"]

Creamos la variable con vs y visualizamos los datos. A su vez, los igualamos a la concatenación de los dataframe de cada pais que hacemos con Panda. Para ello usamos pd.concat Finalmente introducimos axis=1, y así se ordenan los países en una sola fila Escribimos vs para que nos enseñe la concatenacion de los datos que hemos creado

vs = pd.concat([casos_es,casos_it,casos_ger],axis=1)
