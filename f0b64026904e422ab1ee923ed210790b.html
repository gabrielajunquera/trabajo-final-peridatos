Una vez descargada la librería seaborn que dota de una interfaz para mostrar los datos, creamos una dataframe (df)

Es importante importar seaborn como sns, ya que así permitirá cargar los datos del archivo iris

Establecemos el dataframe igual a la visualización de los datos de iris utilizando la librería como sns

El último paso es convertir el df en la linea de texto: sns.kdeplot (df["sepal_width"]) teniendo en cuenta que queremos mostrar en la variable x del gráfico. En el caso que estamos utilizando sería sepal width

# library & dataset
import seaborn as sns
df = sns.load_dataset('iris')

# Make default density plot
sns.kdeplot(df['sepal_width'])
